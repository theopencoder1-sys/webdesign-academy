🗺️ Web Developer Career Roadmap

From Zero to Senior Developer - Your Complete Path

🌱 Phase 1: Beginner (0-3 Months)

📚 Learn

  • ✅ HTML5 Fundamentals
  • ✅ CSS3 & Responsive Design
  • ✅ JavaScript Basics
  • ✅ Git & GitHub
  • ✅ VS Code Mastery

🛠️ Build

  • Personal Portfolio
  • Landing Page
  • Calculator App
  • To-Do List

💰 Earn

  • Freelance: Ksh 5K-15K
  • Intern: Ksh 15K-30K

🌿 Phase 2: Intermediate (3-6 Months)

📚 Learn

  • ✅ Advanced JavaScript (ES6+)
  • ✅ React.js or Vue.js
  • ✅ Tailwind CSS
  • ✅ REST APIs
  • ✅ Databases (SQL)

🛠️ Build

  • E-commerce Site
  • Blog Platform
  • Weather App
  • Chat Application

💰 Earn

  • Junior Dev: Ksh 50K-80K
  • Freelance: Ksh 30K-100K

🔥 Phase 3: Advanced (6-12 Months)

📚 Learn

  • ✅ Full-Stack (Django/Node.js)
  • ✅ Docker & DevOps
  • ✅ System Design
  • ✅ Testing & CI/CD
  • ✅ Cloud (AWS/GCP)

🛠️ Build

  • SaaS Product
  • Mobile App
  • API Service
  • Open Source Project

💰 Earn

  • Senior Dev: Ksh 150K-300K
  • Remote: $3K-$8K/month
