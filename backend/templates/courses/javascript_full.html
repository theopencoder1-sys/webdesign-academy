Chapter 1

What is JavaScript?

JavaScript is the programming language of the web. It is what makes websites interactive, dynamic, and alive. While HTML gives structure and CSS gives style, JavaScript gives behavior.

Created in 1995 by Brendan Eich at Netscape, JavaScript has grown from a simple scripting language to the most popular programming language in the world. Today it runs in every browser, on servers (Node.js), mobile apps (React Native), desktop apps (Electron), and even spacecraft (seriously, NASA uses JavaScript).

💡 Key Insight: JavaScript is NOT related to Java despite the similar name. The name was a marketing decision in the 90s. JavaScript is a completely different language with its own unique features and ecosystem.

Where JavaScript Runs

  • Browser (Frontend): Manipulates web pages, handles user events, fetches data
  • Server (Backend): Node.js runs JavaScript on servers — APIs, databases, file systems
  • Mobile Apps: React Native, Ionic, NativeScript
  • Desktop Apps: Electron (VS Code, Discord, Slack are built with it!)
  • Games: Browser games, Phaser.js, Three.js for 3D
  • Machine Learning: TensorFlow.js runs AI models in the browser

Adding JavaScript to Your Page

<!-- Internal JavaScript -->
<script>
    console.log("Hello from JavaScript!");
</script>

<!-- External JavaScript (BEST PRACTICE) -->
<script src="script.js"></script>

<!-- Always place scripts at the bottom of <body> -->
<!-- Or use defer/async attributes -->
<script src="script.js" defer></script>
Chapter 2

Variables & Data Types

Variables are containers for storing data. JavaScript has three ways to declare variables, but only two you should actually use.

Variable Declaration

// ✅ USE THESE TWO
let name = "Brian";        // Can be reassigned
const PI = 3.14159;        // Cannot be reassigned (constant)

// ❌ NEVER USE THIS
var age = 25;              // Old, problematic, function-scoped
💡 Rule of Thumb: Always start with const. Only use let when you know the value will change. Never use var.

Primitive Data Types

// String — text
let city = "Nairobi";
let country = 'Kenya';
let template = `I live in ${city}, ${country}`; // Template literal

// Number — integers and decimals
let score = 100;
let price = 99.99;
let infinity = Infinity;
let notANumber = NaN; // Result of invalid math

// Boolean — true or false
let isLoggedIn = true;
let isPremium = false;

// Undefined — declared but not assigned
let x; // undefined

// Null — intentionally empty
let y = null;

// Symbol — unique identifier (advanced)
let sym = Symbol("unique");

// BigInt — very large numbers
let big = 9007199254740991n;

Reference Data Types

// Array — ordered list
let colors = ["red", "green", "blue"];
let mixed = [42, "hello", true, null, {name: "Brian"}];

// Object — key-value pairs
let user = {
    name: "Brian",
    age: 25,
    city: "Nairobi",
    isStudent: true,
    skills: ["HTML", "CSS", "JS"]
};

// Accessing properties
console.log(user.name);       // "Brian" (dot notation)
console.log(user["age"]);     // 25 (bracket notation)
console.log(user.skills[0]);  // "HTML"

Type Checking

typeof "Hello"    // "string"
typeof 42         // "number"
typeof true       // "boolean"
typeof undefined  // "undefined"
typeof null       // "object" (this is a bug in JavaScript!)
typeof []         // "object"
typeof {}         // "object"
typeof console.log // "function"
Chapter 3

Operators & Expressions

Arithmetic Operators

let a = 10, b = 3;
console.log(a + b);  // 13 (addition)
console.log(a - b);  // 7  (subtraction)
console.log(a * b);  // 30 (multiplication)
console.log(a / b);  // 3.333... (division)
console.log(a % b);  // 1  (remainder/modulo)
console.log(a ** b); // 1000 (exponentiation)
a++; // Increment (a becomes 11)
b--; // Decrement (b becomes 2)

Comparison Operators

console.log(5 > 3);   // true
console.log(5 < 3);   // false
console.log(5 >= 5);  // true
console.log(5 <= 4);  // false
console.log(5 == "5");  // true (loose equality — converts type)
console.log(5 === "5"); // false (strict equality — checks type too!)
console.log(5 != "5");  // false
console.log(5 !== "5"); // true
💡 ALWAYS use === (strict equality)! == converts types which leads to bugs. === checks both value AND type.

Logical Operators

// AND (&&) — both must be true
console.log(true && true);   // true
console.log(true && false);  // false

// OR (||) — one must be true
console.log(true || false);  // true
console.log(false || false); // false

// NOT (!) — reverses
console.log(!true);  // false
console.log(!false); // true

// Practical examples
let age = 20;
let hasLicense = true;
if (age >= 18 && hasLicense) {
    console.log("You can drive!");
}

let isWeekend = true;
let isHoliday = false;
if (isWeekend || isHoliday) {
    console.log("No work today!");
}
Chapter 4

Conditionals & Control Flow

if / else if / else

let score = 85;

if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else if (score >= 70) {
    console.log("Grade: C");
} else if (score >= 60) {
    console.log("Grade: D");
} else {
    console.log("Grade: F");
}
// Output: "Grade: B"

Ternary Operator (Shortcut)

// Syntax: condition ? valueIfTrue : valueIfFalse
let age = 20;
let canVote = age >= 18 ? "Yes" : "No";
console.log(canVote); // "Yes"

let isLoggedIn = false;
let greeting = isLoggedIn ? "Welcome back!" : "Please login";
console.log(greeting); // "Please login"

Switch Statement

let day = "Monday";

switch(day) {
    case "Monday":
        console.log("Start of the week!");
        break;
    case "Friday":
        console.log("Weekend is near!");
        break;
    case "Saturday":
    case "Sunday":
        console.log("Weekend!");
        break;
    default:
        console.log("Regular day");
}
// Don't forget break! Without it, execution "falls through"
Chapter 5

Loops

For Loop

// Standard for loop
for (let i = 0; i < 5; i++) {
    console.log(`Iteration ${i}`);
}

// Loop through array
let fruits = ["apple", "banana", "mango"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

For...of (Best for Arrays)

let colors = ["red", "green", "blue"];
for (let color of colors) {
    console.log(color);
}

For...in (For Objects)

let user = { name: "Brian", age: 25, city: "Nairobi" };
for (let key in user) {
    console.log(`${key}: ${user[key]}`);
}

While & Do...While

// While loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// Do...while (runs at least once)
let x = 0;
do {
    console.log(x);
    x++;
} while (x < 5);
Chapter 6

Functions Deep Dive

Function Declaration

function greet(name) {
    return `Hello, ${name}!`;
}
console.log(greet("Brian")); // "Hello, Brian!"

Arrow Functions (ES6)

// Full syntax
const add = (a, b) => {
    return a + b;
};

// Short syntax (single expression)
const add = (a, b) => a + b;

// Single parameter (no parentheses needed)
const greet = name => `Hello, ${name}!`;

// No parameters
const sayHello = () => "Hello!";

Parameters & Arguments

// Default parameters
function calculateFee(amount, rate = 0.01) {
    return amount * rate;
}
console.log(calculateFee(1000)); // 10 (uses default rate)

// Rest parameters (...)
function sum(...numbers) {
    return numbers.reduce((total, n) => total + n, 0);
}
console.log(sum(1, 2, 3, 4, 5)); // 15

Callback Functions

// A callback is a function passed as an argument
function processUserInput(name, callback) {
    const greeting = `Hello, ${name}!`;
    callback(greeting);
}

processUserInput("Brian", function(message) {
    console.log(message); // "Hello, Brian!"
});

// With arrow function
processUserInput("Brian", (message) => {
    console.log(message);
});
Chapter 7

DOM Manipulation

The DOM (Document Object Model) is a tree representation of your HTML that JavaScript can interact with.

Selecting Elements

// By ID (returns single element)
const header = document.getElementById("header");

// By class (returns HTMLCollection — like an array)
const buttons = document.getElementsByClassName("btn");

// By tag name
const paragraphs = document.getElementsByTagName("p");

// Query Selector (MOST POWERFUL — returns first match)
const mainTitle = document.querySelector("h1");
const loginButton = document.querySelector(".btn-primary");

// Query Selector All (returns NodeList — all matches)
const allLinks = document.querySelectorAll("a");
const navItems = document.querySelectorAll("nav a");

Changing Content

// Change text
document.querySelector("h1").textContent = "New Title!";

// Change HTML (be careful with this!)
document.querySelector(".content").innerHTML = "<strong>Bold text</strong>";

// Change attributes
document.querySelector("img").src = "new-image.jpg";
document.querySelector("a").href = "https://google.com";

// Change styles
const box = document.querySelector(".box");
box.style.backgroundColor = "red";
box.style.color = "white";
box.style.padding = "20px";
box.style.display = "none"; // Hide element

Creating & Removing Elements

// Create new element
const newDiv = document.createElement("div");
newDiv.textContent = "I am new!";
newDiv.className = "card";
newDiv.id = "myCard";
document.body.appendChild(newDiv);

// Remove element
const oldElement = document.querySelector(".old");
oldElement.remove();

// Insert before another element
const parent = document.querySelector(".container");
const reference = document.querySelector(".reference");
parent.insertBefore(newDiv, reference);
Chapter 8

Events

Adding Event Listeners

// Click event
const button = document.querySelector("button");
button.addEventListener("click", function() {
    alert("Button clicked!");
});

// Arrow function version
button.addEventListener("click", () => {
    console.log("Clicked!");
});

// Form submit
const form = document.querySelector("form");
form.addEventListener("submit", (e) => {
    e.preventDefault(); // Stop page from refreshing
    console.log("Form submitted!");
});

// Input change
const input = document.querySelector("input");
input.addEventListener("input", (e) => {
    console.log("Current value:", e.target.value);
});

// Keyboard events
document.addEventListener("keydown", (e) => {
    console.log("Key pressed:", e.key);
    if (e.key === "Escape") {
        console.log("Escape pressed!");
    }
});

// Mouse events
element.addEventListener("mouseenter", () => {
    element.style.backgroundColor = "blue";
});
element.addEventListener("mouseleave", () => {
    element.style.backgroundColor = "";
});

Common Events List

  • click — Mouse click
  • dblclick — Double click
  • submit — Form submission
  • input — Input value changes
  • change — Select/checkbox changes
  • keydown / keyup — Keyboard keys
  • mouseenter / mouseleave — Hover
  • scroll — Page scrolling
  • resize — Window resizing
  • load — Page finished loading
Chapter 9

Async JavaScript

Callbacks (Old School)

// Callback hell — avoid this!
getUser(1, function(user) {
    getPosts(user.id, function(posts) {
        getComments(posts[0].id, function(comments) {
            console.log(comments);
        });
    });
});

Promises (Modern)

// Creating a promise
const promise = new Promise((resolve, reject) => {
    setTimeout(() => {
        const success = true;
        if (success) {
            resolve("Data loaded!");
        } else {
            reject("Error loading data");
        }
    }, 2000);
});

// Using promises
promise
    .then(data => console.log(data))  // "Data loaded!"
    .catch(error => console.error(error))
    .finally(() => console.log("Done!"));

Fetch API

// GET request
fetch("https://api.example.com/users")
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error("Error:", error));

// POST request
fetch("https://api.example.com/users", {
    method: "POST",
    headers: {
        "Content-Type": "application/json"
    },
    body: JSON.stringify({
        name: "Brian",
        email: "brian@example.com"
    })
})
    .then(response => response.json())
    .then(data => console.log("Created:", data));

Async/Await (The BEST Way)

async function getUsers() {
    try {
        const response = await fetch("https://api.example.com/users");
        const data = await response.json();
        console.log(data);
        return data;
    } catch (error) {
        console.error("Error:", error);
    }
}

// Call the function
getUsers();

🎯 Real-World Project: M-Pesa Calculator

Build a complete M-Pesa fee calculator that takes an amount and calculates the transaction fee.

<!-- index.html -->
<div class="calculator">
    <h2>M-Pesa Fee Calculator</h2>
    <input type="number" id="amount" placeholder="Enter amount (Ksh)">
    <select id="transactionType">
        <option value="send">Send Money</option>
        <option value="withdraw">Withdraw</option>
        <option value="paybill">Pay Bill</option>
    </select>
    <button onclick="calculate()">Calculate</button>
    <div id="result"></div>
</div>

<script>
function calculate() {
    const amount = Number(document.getElementById("amount").value);
    const type = document.getElementById("transactionType").value;
    
    // M-Pesa fee structure (simplified)
    const fees = {
        send: amount <= 100 ? 0 : amount <= 500 ? 7 : amount <= 1000 ? 13 : 
              amount <= 1500 ? 23 : amount <= 2500 ? 33 : amount <= 3500 ? 53 :
              amount <= 5000 ? 67 : amount <= 7500 ? 87 : amount <= 10000 ? 97 :
              amount <= 15000 ? 117 : amount <= 20000 ? 132 : amount <= 25000 ? 152 :
              amount <= 30000 ? 167 : 187,
        withdraw: amount <= 100 ? 10 : amount <= 2500 ? 27 : amount <= 5000 ? 47 :
                  amount <= 10000 ? 67 : amount <= 20000 ? 107 : 167,
        paybill: 0 // Most Paybill transactions are free
    };
    
    const fee = fees[type] || 0;
    const total = amount + fee;
    
    document.getElementById("result").innerHTML = `
        <div class="result-card">
            <p>Amount: <strong>Ksh ${amount.toLocaleString()}</strong></p>
            <p>Fee: <strong>Ksh ${fee.toLocaleString()}</strong></p>
            <p>Total: <strong>Ksh ${total.toLocaleString()}</strong></p>
        </div>
    `;
    
    // Also log to console for debugging
    console.log({amount, type, fee, total});
}
</script>
