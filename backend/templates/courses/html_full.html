🧱 What is HTML?

HTML stands for HyperText Markup Language. It is the foundation of every website you visit — from Google to YouTube to this platform.

Think of HTML as the skeleton of a website. Just like your skeleton gives your body structure, HTML gives structure to web pages. Without it, there would be no websites!

🏗️

HTML = Structure

Like a building's framework

💡 Key Insight:

HTML is NOT a programming language — it doesn't perform calculations or make decisions. It's a markup language that tells browsers "this is a heading, this is a paragraph, this is an image."

📜 History of HTML

1991 — The Beginning

Tim Berners-Lee created HTML at CERN. The first version had only 18 tags. Simple, but revolutionary.

1995 — HTML 2.0

The first official standard. Added forms, tables, and basic structure.

1997-1999 — HTML 3.2 & 4.01

Major updates! Added CSS support, scripting, frames, and better tables.

2014 — HTML5 🎉

The current standard! Added video/audio, semantic elements, canvas, local storage, and much more. This is what we use today.

🏗️ Structure of a Web Page

Every HTML document has a standard structure. Think of it like a letter — it has a specific format that everyone follows.

📄 index.html ✅ This is a complete HTML5 document 
<!DOCTYPE html>  <!-- Tells browser this is HTML5 -->
<html lang="en">        <!-- Root element -->
<head>                <!-- Information ABOUT the page -->
    <meta charset="UTF-8">  <!-- Character encoding -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>  <!-- Browser tab text -->
</head>
<body>                <!-- Everything VISIBLE on the page -->
    <h1>Hello, World! 🌍</h1>
    <p>Welcome to my first website.</p>
</body>
</html>
