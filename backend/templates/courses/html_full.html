🧱 What is HTML?

HTML stands for HyperText Markup Language. It is the foundation of every website you visit — from Google to YouTube to this platform.

Think of HTML as the skeleton of a website. Just like your skeleton gives your body structure, HTML gives structure to web pages. Without it, there would be no websites!

🏗️

HTML = Structure

Like a building's framework

💡 Key Insight:

HTML is NOT a programming language — it doesn't perform calculations or make decisions. It's a markup language that tells browsers "this is a heading, this is a paragraph, this is an image."

📜 History of HTML

1991 — The Beginning

Tim Berners-Lee created HTML at CERN. The first version had only 18 tags. Simple, but revolutionary.

1995 — HTML 2.0

The first official standard. Added forms, tables, and basic structure.

1997-1999 — HTML 3.2 & 4.01

Major updates! Added CSS support, scripting, frames, and better tables.

2014 — HTML5 🎉

The current standard! Added video/audio, semantic elements, canvas, local storage, and much more. This is what we use today.

🏗️ Structure of a Web Page

Every HTML document has a standard structure. Think of it like a letter — it has a specific format that everyone follows.

📄 index.html ✅ This is a complete HTML5 document 
<!DOCTYPE html>  <!-- Tells browser this is HTML5 -->
<html lang="en">        <!-- Root element -->
<head>                <!-- Information ABOUT the page -->
    <meta charset="UTF-8">  <!-- Character encoding -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>  <!-- Browser tab text -->
</head>
<body>                <!-- Everything VISIBLE on the page -->
    <h1>Hello, World! 🌍</h1>
    <p>Welcome to my first website.</p>
</body>
</html>

<!-- ANATOMY OF AN HTML ELEMENT -->

💬 Comments in HTML

Comments are ignored by browsers but help developers understand code. Use them to explain complex sections or temporarily disable code.

<!-- This is a comment. It won't show on the page. -->
<!-- TODO: Add contact form here -->
<!-- FIXME: Image not loading on mobile -->

📝 Text Content Elements - Headings

Headings create document hierarchy. <h1> is most important (use ONCE per page), <h6> is least.

<h1>Main Title - Most Important</h1>
<h2>Section Heading</h2>
<h3>Sub-section</h3>
<h4>Smaller heading</h4>
<h5>Even smaller</h5>
<h6>Smallest heading</h6>
⚠️ SEO Rule: Use exactly ONE <h1> per page. Don't skip levels.

📄 Paragraphs, Line Breaks & Horizontal Rules

<p>This is a paragraph. Browsers add space before and after.</p>
<p>This is another paragraph.</p>
<br>  <!-- Line break - starts new line -->
<hr>  <!-- Horizontal rule - creates a divider -->

🔗 Links and Navigation

The <a> (anchor) tag creates hyperlinks — the foundation of the web!

<!-- External link -->
<a href="https://google.com">Visit Google</a>

<!-- Open in new tab -->
<a href="page.html" target="_blank">New Tab</a>

<!-- Email link -->
<a href="mailto:hello@example.com">Send Email</a>

<!-- Phone link (works on mobile!) -->
<a href="tel:+254700123456">Call Us</a>

🖼️ Images and Media

<!-- Basic Image -->
<img src="photo.jpg" alt="Description">

<!-- Responsive Image -->
<img src="hero.jpg" alt="Banner" style="max-width:100%;height:auto;">

<!-- Video -->
<video controls width="640">
    <source src="video.mp4" type="video/mp4">
</video>

<!-- Audio -->
<audio controls><source src="song.mp3"></audio>
⚠️ ALWAYS use alt text! Screen readers need it. SEO needs it.
