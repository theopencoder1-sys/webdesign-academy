1. What is HTML?

HTML stands for HyperText Markup Language and it is the standard language used to create webpages and web applications. HTML is not a programming language because it does not perform logical operations like loops and conditions but instead structures content on a webpage. Every website on the internet uses HTML in one way or another because browsers understand HTML and display its content visually to users.

HTML works together with CSS and JavaScript where HTML creates the structure, CSS handles the styling and design, while JavaScript adds interactivity and functionality.

🧠 Think of it like building a house:
🏗️ HTML = The structure (walls, roof, foundation)
 🎨 CSS = The design (paint, furniture, decorations)
 ⚡ JavaScript = The functionality (lights, doors, appliances)

2. Basic HTML Structure

An HTML document begins with the declaration <!DOCTYPE html> which tells the browser that the file is written in HTML5. After the declaration comes the <html> tag which acts as the root element containing everything on the webpage. Inside the html tag there are two main sections called the <head> and the <body>.

The head section contains information about the webpage such as the title, links to stylesheets, metadata and scripts while the body section contains everything visible on the webpage like text, images, videos, buttons and forms. A basic HTML structure contains the doctype declaration, html tag, head tag, title tag and body tag. The title tag determines the name shown on the browser tab while the body displays the actual webpage content.

📋 BASIC HTML DOCUMENT
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, Kenya! 🇰🇪</h1>
    <p>Welcome to my first website. I am learning web design!</p>
</body>
</html>
✅ Key Points to Remember:
  • <!DOCTYPE html> must ALWAYS be the first line
  • <html> wraps the entire document
  • <head> = information about the page (not visible)
  • <body> = everything visible to users
  • Save the file as .html and open in any browser

3. HTML Tags & Elements

HTML uses tags to structure content and most tags have an opening tag and a closing tag. For example <p>Hello</p> is a paragraph where <p> is the opening tag and </p> is the closing tag.

Some tags are self-closing or empty tags because they do not need closing tags. Examples include <br> for line breaks, <hr> for horizontal lines, and <img> for images.

Tag Type Example Description
Paired Tag <p>Text</p> Has opening AND closing tag
Self-Closing Tag <br> or <hr> No closing tag needed
Void Element <img src=""> Contains no content, uses attributes

4. Headings & Paragraphs

Headings in HTML range from <h1> to <h6> where <h1> is the largest and most important heading while <h6> is the smallest. Paragraphs are written using the <p> tag and line breaks are created using the <br> tag. Horizontal lines can be added using <hr> to separate sections of content.

📋 HEADINGS & PARAGRAPHS
<h1>Main Heading - Most Important (Use ONLY ONCE per page!)</h1>
<h2>Section Heading</h2>
<h3>Sub-section Heading</h3>
<h4>Smaller Heading</h4>
<h5>Even Smaller</h5>
<h6>Smallest Heading</h6>

<p>This is a paragraph of text. Paragraphs are block elements.</p>
<p>This is another paragraph. Notice the space between them.</p>

<hr>  <!-- Horizontal line for separation -->

<p>This is the first line.<br>This is the second line.</p>
⚠️ Important Rules:
  • Use only ONE <h1> per page (for SEO)
  • Don't skip heading levels (h1 → h2 → h3, not h1 → h3)
  • Headings help with SEO and accessibility

5. Text Formatting

Text formatting can be done using various HTML tags for different purposes.

Tag Effect Usage
<b> Bold text Visual bold only
<strong> Important text Semantic importance (SEO value)
<i> Italic text Visual italic only
<em> Emphasized text Semantic emphasis
<u> Underlined text Underline (use sparingly)
<mark> Highlighted text Highlight for attention
<small> Smaller text Fine print, disclaimers
<del> Deleted text Show removed content
<ins> Inserted text Show added content
<sub> H2O Subscript text

Comments are added using <!-- comment --> and they are ignored by the browser, making them useful for explanations and reminders inside the code.

6. Links & Navigation

Links are one of the most important parts of HTML because they allow navigation between webpages. Links are created using the <a> tag together with the href attribute which stores the destination URL. For example <a href="https://google.com">Google</a> creates a clickable link to Google.

The target attribute with the value _blank can be used to open links in a new tab. HTML can also create email links using mailto: and telephone links using tel:.

🔗 ALL LINK TYPES
<!-- External link -->
<a href="https://google.com">Visit Google</a>

<!-- Open in new tab -->
<a href="https://twitter.com" target="_blank">Twitter (new tab)</a>

<!-- Internal link (same website) -->
<a href="/about.html">About Us</a>

<!-- Jump to section on same page -->
<a href="#section-id">Jump to Section</a>

<!-- Email link -->
<a href="mailto:hello@example.com">Send Email</a>

<!-- Phone link (works on mobile!) -->
<a href="tel:+254700123456">Call Us</a>

<!-- WhatsApp link (Kenya essential!) -->
<a href="https://wa.me/254700123456" target="_blank">Chat on WhatsApp 📱</a>

7. Images

Images are inserted using the <img> tag which requires the src attribute to specify the image path and the alt attribute to describe the image in case it fails to load. Width and height attributes can control image size.

🖼️ IMAGE EXAMPLES
<!-- Basic image -->
<img src="nairobi.jpg" alt="Nairobi city skyline">

<!-- Image with dimensions -->
<img src="logo.png" alt="Company Logo" width="200" height="100">

<!-- Responsive image (BEST for modern websites) -->
<img src="hero.jpg" alt="Banner" style="max-width:100%; height:auto;">

<!-- Image as a clickable link -->
<a href="/about">
    <img src="profile.jpg" alt="Click for about page">
</a>
⚠️ ALWAYS include the alt attribute!
  • Helps visually impaired users (screen readers read it aloud)
  • Shows text if the image fails to load
  • Improves SEO (Google understands your images)

8. Lists

Lists in HTML are used to organize content and there are three main types: ordered lists, unordered lists, and description lists.

📝 ALL LIST TYPES
<!-- Unordered List (bullet points) -->
<ul>
    <li>Nyama Choma</li>
    <li>Ugali</li>
    <li>Chapati</li>
</ul>

<!-- Ordered List (numbered) -->
<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>

<!-- Description List -->
<dl>
    <dt>HTML</dt>
    <dd>Structure of web pages</dd>
    
    <dt>CSS</dt>
    <dd>Style and layout</dd>
</dl>

<!-- Nested List -->
<ul>
    <li>Frontend
        <ol>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ol>
    </li>
</ul>

Ordered lists use the <ol> tag and automatically number items. Unordered lists use the <ul> tag and display bullet points. Each item inside a list is written using the <li> tag. Description lists use <dl>, <dt> and <dd> to define terms and descriptions. Lists can also be nested inside one another to create sublists.

9. Tables

Tables are used in HTML to display data in rows and columns. A table begins with the <table> tag and rows are created using <tr>. Table headings use <th> while normal data cells use <td>.

The colspan attribute allows a cell to span across multiple columns while rowspan allows a cell to span multiple rows. Tables are commonly used for timetables, records, product listings and reports though they should not be used for webpage layout in modern web development.

📊 COMPLETE TABLE EXAMPLE
<table border="1">
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>City</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Brian</td>
            <td>25</td>
            <td>Nairobi</td>
        </tr>
        <tr>
            <td>Mary</td>
            <td>22</td>
            <td>Mombasa</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">Total Students: 2</td>
        </tr>
    </tfoot>
</table>

10. Forms

Forms are essential in HTML because they allow users to input and submit data. The <form> tag acts as a container for form elements such as text boxes, buttons and dropdown menus. The action attribute specifies where form data is sent while the method attribute determines how the data is submitted, commonly using GET or POST.

Input fields are created using the <input> tag and different types are available such as text, password, email, number, date, checkbox and radio. Text fields allow users to type information while password fields hide entered characters. Radio buttons allow one option selection from a group while checkboxes allow multiple selections. Textareas are used for longer text input while dropdown menus are created using <select> and <option> tags.

Labels improve accessibility by describing form fields. Buttons can submit forms using either <button> or <input type="submit">. HTML5 introduced advanced form validation where attributes such as required, minlength, maxlength and placeholder help improve user experience and reduce invalid submissions.

📋 COMPLETE FORM EXAMPLE
<form action="/submit" method="POST">
    
    <label for="name">Full Name:</label>
    <input type="text" id="name" name="name" placeholder="Brian Ochieng" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" placeholder="brian@example.com" required>
    
    <label for="phone">Phone Number:</label>
    <input type="tel" id="phone" name="phone" placeholder="0712345678">
    
    <label for="course">Course:</label>
    <select id="course" name="course">
        <option value="">-- Select --</option>
        <option value="html">HTML5</option>
        <option value="css">CSS</option>
        <option value="js">JavaScript</option>
    </select>
    
    <fieldset>
        <legend>Payment Method:</legend>
        <input type="radio" id="mpesa" name="payment" value="mpesa" checked>
        <label for="mpesa">M-Pesa</label>
        
        <input type="radio" id="card" name="payment" value="card">
        <label for="card">Bank Card</label>
    </fieldset>
    
    <input type="checkbox" id="terms" name="terms" required>
    <label for="terms">I agree to terms and conditions</label>
    
    <label for="message">Message:</label>
    <textarea id="message" name="message" rows="4" placeholder="Your message..."></textarea>
    
    <button type="submit">Submit</button>
    
</form>

11. Semantic HTML

Semantic HTML refers to the use of tags that clearly describe their meaning and purpose. Semantic tags make code easier to read, improve accessibility and help search engines understand webpage structure.

Examples include <header> for the top section of a page, <nav> for navigation links, <main> for the main content, <section> for grouped content, <article> for independent articles, <aside> for sidebars and <footer> for the bottom section of the page. Using semantic HTML instead of excessive <div> tags results in cleaner and more professional code.

✅ SEMANTIC (Good) 
<header>
    <nav>...</nav>
</header>
<main>
    <article>...</article>
</main>
<footer>...</footer>
❌ NON-SEMANTIC (Bad) 
<div class="header">
    <div class="nav">...</div>
</div>
<div class="main">
    <div class="article">...</div>
</div>
<div class="footer">...</div>

12. Multimedia

HTML also supports multimedia through audio and video tags. The <audio> tag can embed sound files while the <video> tag can embed videos directly into webpages. Both support controls such as play, pause and volume adjustment. YouTube videos can be embedded using <iframe> tags.

🎬 MULTIMEDIA EXAMPLES
<!-- Audio -->
<audio controls>
    <source src="song.mp3" type="audio/mpeg">
    Your browser does not support audio.
</audio>

<!-- Video -->
<video width="640" height="360" controls>
    <source src="tutorial.mp4" type="video/mp4">
    Your browser does not support video.
</video>

<!-- YouTube Embed -->
<iframe width="560" height="315" 
    src="https://www.youtube.com/embed/VIDEO_ID" 
    title="YouTube video" 
    frameborder="0" 
    allowfullscreen>
</iframe>

HTML entities are special codes used to display reserved characters such as &lt; for less than (<), &gt; for greater than (>) and &copy; for the copyright symbol (©). Entities help avoid conflicts with actual HTML tags.

13. HTML Attributes

HTML attributes provide additional information about elements and are written inside opening tags. Common attributes include id, class, style, src, href, alt, width, height and title. The id attribute uniquely identifies an element while class groups multiple elements together for styling. Inline CSS can be added using the style attribute although external CSS is preferred for larger projects.

HTML5 introduced many modern features such as semantic tags, audio, video, local storage and improved forms. HTML is the foundation of web development and understanding it properly is important before learning CSS and JavaScript because every webpage structure depends on HTML.

✅ HTML Best Practices

Clean HTML code should be properly indented, use meaningful tags, include comments where necessary and follow good structure practices to improve readability and maintenance.

✅ DO THIS
  • Use semantic HTML tags
  • Always include alt text for images
  • Use proper heading hierarchy
  • Include viewport meta tag
  • Close all tags properly
  • Use lowercase tag names
  • Indent nested elements
  • Add helpful comments
❌ AVOID THIS
  • Skip heading levels
  • Use divs for everything
  • Forget to close tags
  • Use deprecated tags (<center>, <font>)
  • Inline styles everywhere
  • Forget mobile responsiveness
  • Messy indentation
  • Missing alt attributes
