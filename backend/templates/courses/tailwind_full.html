1. Why Tailwind CSS?

Tailwind CSS is a utility-first CSS framework that lets you build modern websites directly in your HTML without writing custom CSS. Instead of naming classes, you use pre-built utility classes.

✅ With Tailwind 
<button class="bg-blue-500 
  text-white 
  px-6 py-3 
  rounded-lg 
  hover:bg-blue-600">
  Click Me
</button>
❌ Without Tailwind 
<button class="btn-primary">
  Click Me
</button>

/* Separate CSS file */
.btn-primary {
  background: blue;
  color: white;
  padding: 12px 24px;
  border-radius: 8px;
}
.btn-primary:hover {
  background: darkblue;
}

2. Setup & Installation

Quick Start (CDN - Easiest!)

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <script src="https://cdn.tailwindcss.com"></script>
</head>
<body>
    <h1 class="text-3xl font-bold text-blue-600">
        Hello Tailwind!
    </h1>
</body>
</html>

3. Colors & Typography

<!-- Text Colors -->
<p class="text-red-500">Red text</p>
<p class="text-blue-600">Blue text</p>
<p class="text-green-400">Green text</p>

<!-- Background Colors -->
<div class="bg-gray-800">Dark background</div>
<div class="bg-indigo-500">Indigo background</div>

<!-- Typography -->
<h1 class="text-4xl font-bold">Big Bold Heading</h1>
<p class="text-lg font-medium">Medium paragraph</p>
<p class="text-sm text-gray-500">Small gray text</p>

<!-- Font weights -->
<span class="font-light">Light</span>
<span class="font-normal">Normal</span>
<span class="font-semibold">Semi Bold</span>
<span class="font-bold">Bold</span>
<span class="font-black">Black</span>

4. Spacing & Sizing

<!-- Padding (p-) -->
<div class="p-4">Padding all sides</div>
<div class="px-6 py-3">Horizontal & vertical</div>
<div class="pt-8 pb-4 pl-2 pr-2">Each side</div>

<!-- Margin (m-) -->
<div class="m-4">Margin all sides</div>
<div class="mx-auto">Center horizontally!</div>
<div class="mt-8">Margin top only</div>

<!-- Width & Height -->
<div class="w-full">Full width</div>
<div class="w-1/2">Half width</div>
<div class="h-screen">Full screen height</div>
<div class="max-w-4xl mx-auto">Max width + centered</div>

5. Flexbox & Grid

<!-- Flexbox -->
<div class="flex justify-center items-center gap-4">
    <div>Item 1</div>
    <div>Item 2</div>
</div>

<!-- Perfect Centering -->
<div class="flex justify-center items-center h-screen">
    <h1>Perfectly Centered!</h1>
</div>

<!-- Grid -->
<div class="grid grid-cols-3 gap-4">
    <div>Col 1</div>
    <div>Col 2</div>
    <div>Col 3</div>
</div>

<!-- Responsive Grid -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
    <!-- Cards go here -->
</div>
